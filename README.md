## Day 3 (2021-07-10)

为什么要用Generic Views呢？感觉一开始不用也很好啊。Django tutorials一开始没有使用Generic Views我觉得挺好，而是用最自然最原始的一种方法处理url，让你可以快速掌握url, view & template这些概念和应用，而不是花时间去适应和深究Generic View这种模版。

Django的好处之一就是可以自动生成url，可以随意调用。
